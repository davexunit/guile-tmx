guile-tmx
=========

Tiled .tmx file parser for GNU Guile.

About
-----

guile-tmx parses map files produced by the
[Tiled](http://www.mapeditor.org/) 2D tile map editor.

It *only* parses. There are no dependencies on graphics/game
libraries.
